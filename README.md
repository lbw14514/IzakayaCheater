# IzakayaCheater
东方夜雀食堂修改器，支持金钱修改和存档修改。

## 功能

### 金钱修改
运行时修改游戏内存中的金钱。需先启动游戏，点击"检测进程"附加后修改。

### 存档修改
直接修改游戏存档文件（JSON 格式），无需启动游戏。

**添加邀请函**
- 向存档添加 DLC2 怪诞料理邀请函（物品 ID 2014~2019）
- 自动将 DLC2 六位角色好感度设为满级

**触发博丽大祭**
- DLC3 六位角色好感度满级
- 设置博丽祭相关开关
- 完成羁绊进度任务
- 注意：此操作可能跳过部分剧情

## 支持版本
4.1.1+

## 编译（仅 Windows）

### 环境配置
- C/C++ 编译器（推荐 MinGW GCC）
- CMake >= 3.5.0
- wxWidgets >= 3.2（MSW 版本，用同一编译器编译）

### 编译
```bash
mkdir build && cd build
cmake .. -G "Ninja" -DCMAKE_BUILD_TYPE=Release
ninja
```

### 运行
将 `DLLS/` 目录下的 `.dll` 文件复制到 exe 同目录，双击运行。
